## 股票系统 (stock_api.py)

### 已修复的问题
1. **搜索框失效** — 新增 `/api/search` 接口，支持按名称/代码模糊搜索全量股票
2. **每日复盘日期undefined** — `trade_date` 路径错误（`m.trade_date` → `d.data.trade_date`）
3. **上证收盘无数据** — 字段名错误（`index_close` → `shanghai_close`）
4. **10日最高点数据错误** — SQL 列索引错用 `r[5]→r[3]`，r[5]是volume不是high
5. **估值进度条超屏** — bar百分比上限100%，颜色判断逻辑错误（70/20=3.5永远>0.6）
6. **pct_change全部为0** — 回填脚本用错公式`(close-open)/open`，应`(今收-昨收)/昨收*100`；已回填修正33万行
7. **连涨/连跌天数计算错误** — 应比较相邻两天收盘价，而非与今日比较；修正算法
